# Mnemos

> Company Brain — capture how your company works, make it executable for AI.

A macOS overlay app that lets teams record knowledge snippets throughout the day. The captured knowledge becomes a living, structured skills file that AI agents can act on.

## Status
Early MVP — overlay + ephemeral local capture.

## Structure
- `.claude/` — Claude Code commands, hooks, project settings
- `topology/` — service map, data models, async contracts
- `.github/` — CI workflows, issue/PR templates
